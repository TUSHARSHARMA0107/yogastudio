# 🧘 Yoga Studio Website

A modern and responsive **Yoga Studio Website** designed to showcase studio facilities, yoga programs, trainer information, and user experiences through an engaging and visually calming interface.

The website provides visitors with detailed information about yoga classes, studio environment, and training philosophy while offering an intuitive and visually appealing browsing experience.

---

## ✨ Features

- 🧘 Modern and responsive UI
- 📸 Studio tour with images and videos
- 📚 Programs and yoga classes sections
- 👩‍🏫 Trainer introduction page
- ⭐ Custom reviews and testimonials
- 📝 Blog / vlog integration
- 💬 WhatsApp session booking integration
- 🎨 Smooth animations and transitions
- 📱 Fully responsive design

---

## 🛠️ Technologies Used

- **React.js (Vite)** – Frontend framework
- **Tailwind CSS** – Styling and responsive design
- **Framer Motion** – Animations and transitions
- **React Router** – Multi-page navigation
- **Lucide React Icons** – Icon library
